# Building-own-project
The Burnout Tracker is a simple Python-based modd and stress monitoring tool that asks you basic questions, stores data for the week and generates a weekly mood graph using Matplotlib.
It is useful for identifying stress patterns and maintaining emotional awareness.
Features:- 
 - record weekly mood,
 - generates a weekly visualisation plot using matplotlib
 - track patterns over time
 - reflect and heal slowly
It is perfect for students or anyone who wants a safe space to observe their emotional well-being.
